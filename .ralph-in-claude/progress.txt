# Progress Log — I2C Demo Platform

## Codebase Patterns
- Verilog 8-bit register arithmetic naturally wraps (255 + 1 = 0), no explicit modulo needed
- reg_addr was already 8-bit in original design
- FastAPI app uses lifespan context manager pattern; CORS allows all origins for dev
- mypy with --ignore-missing-imports for typecheck (fastapi has stubs, cocotb does not)
- backend/sim needs __init__.py for pytest imports (from sim.X import Y)
- Frontend worktrees need `bun install` before tsc; NodePaletteItem helper used in Sidebar
- Node type strings: 'i2c_start', 'i2c_stop', 'repeated_start' to avoid RF internal conflicts
- vcdvcd timescale: dict with magnitude (Decimal), unit (str); use _UNIT_TO_PS lookup for ps conversion
- cocotb get_sim_time uses positional 'ps' arg (not units= keyword) in cocotb 2.0
- Horizontal layout: applyHorizontalLayout + buildAutoEdges helpers in App.tsx
